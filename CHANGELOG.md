# Changelog

All notable changes to Submail are documented here.

## [0.1.0] - 2026-07-10

Initial public release.

- Unified multi-mailbox Web inbox with IMAP/POP3 sync and SMTP sending.
- Drafts, conversations, search, aliases, attachments, and local message state.
- OpenAI-compatible summaries, reply suggestions, composition, and translation.
- Local stdio and remote Streamable HTTP MCP with scoped API keys.
- Direct HTTP send API with idempotency support.
- Docker deployment with Redis/BullMQ and SQLite, bundled MySQL, or external MySQL.
- SQLite online backup, atomic restore, health checks, retention policies, and integration tests.
